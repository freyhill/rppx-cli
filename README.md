# rppx-cli

a cli used to install react-multi-page-app project


## install

```
$ npm install rppx-cli -g
```

## create 

```
$ rppx init myproject
```

## dev&build

```
$ cd myproject

$ npm install

$ npm run dev //development

$ npm run build // production
```

## start

```
$ npm start
```
