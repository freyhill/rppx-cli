# wrmp-cli

a cli used to install webpack-react-multi-page project


## install

```
npm install wrmp-cli -g

```

## create 

```
wrmp init myproject
```

## dev&build

```
cd myproject

npm install

npm run dev //development

npm run build // production
```

## start

```
npm start
```
